Functions

  1. Create an "isEven()" function
    • takes a single numeric arguement and returns true if the number is even, and false otherwise
  2. Create a "factorial()" function
    • takes a single numeric argument and returns the factorial of that number
  3. Create a "kebabToSnake()" function
    • takes a single kebab-cased string argument and returns the snake_cased version. Basically replace "-"s to "_"s
